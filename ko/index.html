Most Popular

AI CLI 모니터링

Claude Code, Cursor, Copilot CLI가 코드를 생성하는 동안 모바일에서 실시간으로 진행 상황 확인

claude "전체 테스트 실행하고 실패하면 수정해줘" → 커피 마시면서 폰으로 확인

장시간 빌드 모니터링

CI/CD 파이프라인, Docker 빌드, 대규모 컴파일 작업을 어디서든 체크

배포 상태 확인

서버 배포 중 이동해야 할 때, 폰으로 로그 확인하고 문제 발생 시 즉시 대응

페어 프로그래밍

동료에게 내 터미널을 공유하여 원격으로 함께 디버깅하고 코드 리뷰

주요 기능

끊김 없는 터미널 경험을 위한 기술

실시간 스트리밍

WebSocket 기반 50ms 이하 지연으로 AI CLI 출력을 실시간으로 확인

모바일 최적화

스마트폰 브라우저에서도 완벽하게 동작하는 반응형 터미널 UI

안전한 접속

Google OAuth2 인증으로 본인의 세션만 안전하게 접근

양방향 입력

모바일에서도 키보드 입력 가능, AI에게 추가 지시를 내리세요

멀티 세션

여러 터미널 세션을 동시에 관리하고 손쉽게 전환

데이터 압축

GZip 압축으로 모바일 데이터 사용량 최소화
