Stream Your Terminal
Anywhere, Anytime

SessionCast는 tmux 세션을 실시간으로 스트리밍하여 어디서든 웹 브라우저로 터미널에 접근할 수 있게 해주는 오픈소스 원격 터미널 솔루션입니다.

Get Started View on GitHub
sessioncast — tmux 
$ tmux new -s workspace
# Creating new tmux session...

$ sessioncast-agent start
✓ Connected to relay server
✓ Session 'workspace' registered
→ Streaming to: https://your-server.com

$ _

Features

강력하고 유연한 원격 터미널 스트리밍

Real-time Streaming

WebSocket 기반의 실시간 화면 전송으로 지연 없는 터미널 경험을 제공합니다.

Web-based Access

별도의 클라이언트 설치 없이 웹 브라우저만으로 어디서든 터미널에 접근하세요.

Secure Authentication

Google OAuth2와 JWT 토큰 기반 인증으로 안전하게 세션을 보호합니다.

GZip Compression

화면 데이터를 압축하여 네트워크 대역폭을 절약하고 빠른 전송을 보장합니다.

Multi-session Support

여러 tmux 세션을 동시에 관리하고 손쉽게 전환하세요.

Collaborative Viewing

여러 사용자가 동시에 같은 세션을 시청할 수 있어 협업과 교육에 적합합니다.

Architecture

간단하면서도 확장 가능한 3-tier 아키텍처

Agent

tmux 세션 캡처 및 전송

Java / Spring Boot
WebSocket

Relay Server

세션 중계 및 인증 관리

Spring Boot / WebSocket
WebSocket

Web Client

브라우저 기반 터미널 UI

React / xterm.js

Quick Start

5분 안에 시작하기

1

Relay Server 배포

docker pull sessioncast/server:latest docker run -p 8080:8080 sessioncast/server
2

Agent 설치 및 설정

curl -L https://sessioncast.io/install.sh | bash sessioncast-agent config --relay wss://your-server/ws
3

웹에서 접속

https://your-server.com

Google 계정으로 로그인 후 세션 목록에서 선택
